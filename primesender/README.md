## 🚀 Overview
**PrimeSender** is a high-performance web utility designed for marketers, organizers, and anyone dealing with large contact lists. It eliminates the manual headache of cleaning strings and formats, allowing you to go from a raw PDF report to a WhatsApp-ready contact list in seconds.

## ✨ Key Features

### 🔍 Intelligent Extraction
- **Text Parsing:** Paste raw, messy text; it automatically finds and extracts valid phone numbers.
- **Excel Support (`.xlsx`, `.csv`):** Upload your spreadsheets, and the tool intelligently identifies columns like 'Phone', 'Mobile', or 'Whatsapp'.
- **PDF Extraction:** Built-in PDF.js integration to scan and pull numbers directly from PDF documents.

### 🛠️ Smart Utilities
- **.vcf Export:** Download all cleaned contacts as a VCF file to instantly import into your phone's address book.
- **Direct WhatsApp Integration:** Send messages or group invite links directly to any extracted number without saving it first.
- **Live Preview:** Real-time badge tracking of unique contacts found.

### 🎨 Premium UI/UX
- **Dark Mode First:** A carefully crafted Slate-900 theme that's easy on the eyes.
- **Responsive Design:** Optimized for both mobile and desktop usage.
- **Interactive Feedback:** Smooth animations and toast notifications for a premium feel.

## 📖 How to Use
1. **Input Data:** Paste your numbers into the text box OR click **"Upload Excel/PDF"**.
2. **Process:** Click **"Process & Clean"** to extract unique, valid numbers.
3. **Action:**
   - Use **"Download .vcf File"** for bulk saving.
   - Attach a **Group Link** and click the WhatsApp icon next to a number to send an invite.
