# BattleHUB 🎮

A sleek, all-in-one **Esports Management System** for tournament organizers.

## ✨ Core Features
- **Rule Maker:** Generate professional, themed rulebooks (Neon, Pro, Crimson) and export them as PDFs.
- **Result Engine:** Calculate points automatically via manual entry or OCR scan (simulated).
- **Customizable:** Fully adjustable point systems, map pools, and game settings.
- **Premium UI:** Modern gaming aesthetic with glassmorphism and neon animations.

## 🛠️ Tech Stack
- **HTML5 & Tailwind CSS**
- **Lucide Icons**
- **jsPDF/html2pdf.js** (PDF Export)
- **Tesseract.js** (OCR Data Extraction)

## 🚀 Quick Start
1. Open `index.html` in any browser.
2. Choose **Rule Maker** to design rulebooks.
3. Choose **Result Engine** to generate live leaderboards.
