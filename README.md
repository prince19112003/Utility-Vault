**A curated collection of professional-grade web utilities designed for productivity, entertainment, and automation.**

Welcome to **Utility-Nexus**, a centralized hub for various web applications that solve niche problems—ranging from AI-powered quiz generation to bulk contact management.

## 🎨 Design Philosophy
Every project in this collection follows a **Premium Dark Aesthetic**:
- **Professional Typography:** Primarily using *Inter* or *Outfit* for maximum legibility.
- **Glassmorphism:** Modern UI components with subtle transparency and border effects.
- **Micro-Animations:** Smooth transitions and interactive hover states.
- **Dark Mode First:** Optimized for productivity and eye comfort.

## 🛠️ Global Technology Stack
This repository leverages the best of modern web development:
- **Frameworks:** React 19, Vite
- **Styling:** Vanilla CSS, Tailwind CSS
- **AI/ML:** Groq Cloud (Llama 3.3), Tesseract.js (OCR)
- **APIs:** Google Forms API, Firebase
- **Data:** SheetJS, jsPDF, PDF.js

---

## 🚀 Getting Started
Most projects are standalone and can be run by simply opening the `index.html` file in your browser. 

For **AI Quiz Automator**:
1. Navigate to `Assigment-Form`
2. Run `npm install`
3. Add your `VITE_GROQ_API_KEY` to a `.env` file.
4. Run `npm run dev`
